# ZMK Configuration for Corne Keyboard

![](https://github.com/thrly/thrly-corne-zmk/actions/workflows/build.yml/badge.svg)

## Overview

- This will almost certainly be a work in progress.
- I got a ton of inspiration from [Thrly's repository](https://github.com/thrly/thrly-corne-zmk)
- Most of this was done using [Nick Coutsos's Keymap Editor](https://nickcoutsos.github.io/keymap-editor/).
- The keymap visualization was made using the [Keymap Drawer](https://nickcoutsos.github.io/keymap-editor/).
- To get the latest firmware, go to the last successful run in the **Actions** tab.
